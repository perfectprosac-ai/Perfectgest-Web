Política de Privacidade — PerfectGest I
Última atualização: Maio de 2026
A PerfectGest, sob responsabilidade de Marcos Leandro dos Santos, estabelece esta Política de Privacidade em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/18) e as políticas para desenvolvedores da Google Play Store.
Controlador de Dados e Contato O responsável pelo tratamento de dados é Marcos Leandro dos Santos. Para o exercício de direitos de acesso, retificação ou exclusão, entre em contato via e-mail: sac.perfectgest@gmail.com
Armazenamento Local e Natureza dos Dados O aplicativo opera em modelo local-first. Os dados de gestão comercial (registros de clientes, orçamentos, fluxos de caixa e projetos) são armazenados exclusivamente no dispositivo do usuário. Não coletamos, acessamos ou transferimos esses dados sensíveis para servidores. A saída desses dados ocorre apenas por ação deliberada do usuário, como a geração de backups ou exportação de PDFs.
Coleta de Dados e Serviços de Terceiros Para garantir a integridade técnica e licenciamento, o aplicativo utiliza serviços da Google LLC (Firebase), que coletam apenas dados estritamente necessários:
Google Analytics para Firebase: Coleta eventos de uso (ex: abertura de telas, erros críticos e atualizações) para análise de estabilidade. Não há coleta de dados nominais ou valores financeiros.
Firebase Remote Config: Utilizado para buscar parâmetros de configuração, como versões mínimas e flags de segurança (kill switch).
Cloud Firestore: Utilizado para persistência de metadados técnicos e controle de período de avaliação (trial). O sistema registra o marco temporal de instalação vinculado a um identificador anônimo do dispositivo (Android ID ou Identifier for Vendor), sem vincular a contas de e-mail ou identidades pessoais.
Google Play Billing: Processa transações financeiras de assinaturas diretamente pela infraestrutura do Google.
Declarações de Segurança e Transparência
Criptografia: Todos os dados em trânsito (telemetria e metadados técnicos) são protegidos via protocolo HTTPS/TLS.
Não Compartilhamento: A PerfectGest declara que não vende ou compartilha dados técnicos ou de utilização com terceiros para fins publicitários.
Finalidade Específica: O uso de identificadores de dispositivo serve exclusivamente para validar o tempo de uso gratuito e garantir a segurança do software.
Retenção e Exclusão de Dados O usuário possui controle total sobre suas informações:
Dados Comerciais: São eliminados instantaneamente ao limpar o cache do aplicativo ou desinstalar o software.
Dados de Telemetria: O usuário pode solicitar a exclusão de registros de suporte ou dúvidas técnicas via e-mail, com prazo de atendimento de até 15 dias úteis.
Permissões de Sistema O aplicativo solicita acesso ao armazenamento apenas para:
Importar logotipos para personalização de documentos.
Salvar relatórios em PDF e arquivos de backup gerados pelo usuário.
Elegibilidade e Foro O aplicativo é destinado a usuários maiores de 18 anos. Esta política é regida pelas leis da República Federativa do Brasil, sendo eleito o foro da comarca de Caxias do Sul/RS para dirimir eventuais controvérsias.
